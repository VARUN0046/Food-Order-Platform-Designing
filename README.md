# Swiggy UI Redesign – Figma Design

## 📌 Project Overview
This project is a UI/UX design created in Figma, inspired by a modern food delivery application experience. The design focuses on creating an intuitive, visually appealing, and user-friendly interface for browsing restaurants, ordering food, and discovering offers.

## 🎯 Objectives
- Create a clean and modern mobile UI.
- Improve user navigation and accessibility.
- Design engaging food discovery sections.
- Follow standard UI/UX design principles.
- Develop both Low-Fidelity and High-Fidelity prototypes.

## 🛠️ Tools Used
- Figma
- UI/UX Design Principles
- Wireframing
- Prototyping

## 📱 Screens Included
- Home Screen
- Food Categories
- Top Rated Restaurants
- Recommended Restaurants
- 99 Store Section
- Bottom Navigation Bar

## 🎨 Design Process
### Low-Fidelity Design
Created wireframes to establish layout structure, content hierarchy, and user flow.

### High-Fidelity Design
Enhanced the wireframes with:
- Color palette
- Typography
- Icons
- Images
- Interactive UI elements
